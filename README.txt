# MONOCHROME ENGINE

Motor de RPG de texto estilo retro 80s.

## Demo incluido

- Demo: `demo_game/scenes.json`
- Ejecutar: `python engine.py`

## Cómo crear tu juego

1. Crear un JSON con tus escenas.
2. Crear un diccionario de enemigos si los usás.
3. Opcional: definir tus propios items y funciones para efectos especiales.
