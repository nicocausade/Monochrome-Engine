# Monochrome-Engine
Motor de RPG de texto estilo retro con combate, enemigos, items y escenas JSON.

Monochrome Engine es un motor de RPG de texto inspirado en los juegos retro de los 80s. 
Permite crear historias interactivas con:

- Sistema de combate por turnos
- Enemigos y NPCs
- Inventario y uso de objetos
- Escenas y eventos definidos en JSON
- Guardado y carga del progreso del jugador
- Opciones condicionales y eventos con probabilidades

Ideal para desarrollar juegos de texto personalizados, demos o experimentos narrativos.
